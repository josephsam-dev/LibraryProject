# LibraryProject

This is the starter Django project for the ALX 'Introduction to Django Development Environment Setup' task.

## Project Overview
The project was created using:
```bash
django-admin startproject LibraryProject
```

It contains:
- **manage.py** — Django's command-line utility.
- **settings.py** — Configuration file for the project.
- **urls.py** — URL routing configuration.
- **wsgi.py** and **asgi.py** — Deployment entry points for web servers.

## How to Run
1. Activate the virtual environment (if created):
   ```bash
   source .venv/Scripts/activate
   ```
2. Start the development server:
   ```bash
   python manage.py runserver
   ```
3. Open your browser and visit http://127.0.0.1:8000/
